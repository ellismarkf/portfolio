projects

what I've done

Wedding RSVP site

You know you're a developer when you write your own wedding RSVP program and use Google's Sheets API to turn a Google Drive spreadsheet into a free database. This project also needed to support i10n, since my fiancee's family only speaks Spanish.


TECHNOLOGIES: ES6, react, babel, webpack, Google Sheets API, redux, react-intl, SSL

Matchsticks

Multiplayer nim game, using websockets to push notifications to every client connected. Built as an exploration of React, websockets, and HTML5 gaming.


TECHNOLOGIES: react, websockets, node, express

AirDrum

JS web-based motion-sensing virtual drum kit, using two Myo Armbands and a Leap Motion Sensor. Built in 24 hrs. Won 2nd place at CodeRED MLH hackathon, Spring 2015. My first project using external hardware as part of the development process.


TECHNOLOGIES: javascript, websockets, Leap Motion Sensor, Myo Armband, grunt, LESS

Lark

Destination-agnostic airfare scanner. A week-long hackathon project that takes a different approach to travel, which explored and tested the limits of RESTful fligtht APIs. Try clicking 'Test' for full simulated results.


TECHNOLOGIES: html5, scss, angular, node, grunt, QPX express api, vanilla js

Transliteratr

An experiment in new visual languages. "Translates" inputted text (poems, quotes, exerpts from novels, etc) into mosaics of randomly generated colors.


TECHNOLOGIES: html5, sass, sinatra, ruby, yaml, erb

Trifecta Training

Built for a basketball training consultancy in Boston, with a focus on contemporary design. This site was my first freelance project, and marks the point in my career when I decided to pursue development full-time.


TECHNOLOGIES: html5, css3, javascript, php

Carltech Engineering

Where it all began. I volunteered to make this website with no prior knowledge or experience, and built it in two months. It represents the commitment to learning and exploring that continues to drive me now.


TECHNOLOGIES: html, css, jQuery
